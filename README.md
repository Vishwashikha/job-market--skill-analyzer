# Job Market Skill Analyzer

## Project Overview
This project analyzes data science job postings to identify important skills, job roles, and trends in the job market.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data cleaning
- Job title analysis
- Industry and sector analysis
- Skill demand analysis
- Visualization of job trends

## Key Insights
- Data Scientist is the most common job role
- Python is the most in-demand skill
- Excel is also widely used
- Spark and Hadoop are important for big data roles

## Conclusion
This project helps understand current job market trends and guides learners on which skills to focus on.
